# AI PM

An experimental AI workflow system for assisting Product Managers with PRD generation and evaluation.
## Current Workflow

PRD Writing Skill -> PRD Agent -> PRD Evaluation Layer

## Goals

- Explore reusable AI skills
- Design agentic PM workflows
- Improve AI-generated PRD quality
- Experiment with evaluation systems

## Repository Structure

- skills/ → reusable AI capabilities
- agents/ → agent definitions
- evaluations/ → quality checks
